## MVP 版本声明
**ManuScript MVP（面试 Demo 版）**
- 目标：证明“文献可追溯检索 + 笔记检索”的可信闭环
- 核心流程：本地导入 PDF → 解析为 Markdown → 全文检索 → 段落级溯源
- 交互能力：对话式检索 + 会话摘要 + 笔记管理
- 成功标准：检索命中率高、溯源准确、可解释性强
- 暂不覆盖：端到端写作生成、博士级深度分析、全量外部搜索
